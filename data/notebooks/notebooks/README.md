# Phoenix Real Estate Investment Analyzer

This project analyzes housing data and builds machine learning models to predict house prices.

## Features
- Data exploration and visualization
- Linear Regression model
- Random Forest model
- Model evaluation using RMSE
- Saved trained model for future predictions

## Technologies
- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib

## Project Structure
data/        - raw dataset  
notebooks/   - analysis scripts  
models/      - saved ML models  
reports/     - charts and outputs  
src/         - reusable code
